Definición de Curva de Nivel

Una curva de nivel de una función de dos variables $z = f(x,y)$ es el conjunto de puntos $(x,y)$ en el dominio donde la función toma un valor constante $k$:

\[f(x,y) = k\]

Geometricamente, es la curva resultado de intersectar la superficie $z = f(x,y)$ con el plano horizontal $z = k$, proyectada sobre el plano $XY$.

Proyección al Plano $z = 0$

Las líneas de trazos muestran cómo cada contorno a la altura $z = k$ se proyecta directamente sobre el plano $XY$, formando el mapa topográfico de curvas de nivel etiquetadas.
